# 私人管理系统
为用户提供事务和日记的管理和提醒，记录开支，收纳备忘录等功能
基于小程序的客户端，Vue的后台管理端，使用Python的Flask框架做后台API接口服务
## 目录
├── client	// 小程序  
├── backstage	// 后台管理（Vue）  
├── server	// 后台（Flask+uWSGI+Nginx）  
├── screen	// 截图
